\frametitle{Git: Workflow}

\begin{multicols}{2}
Git has three main states that your files can reside in:

\begin{itemize}[<+(1)->]
\item \textbf{Modified}
\item \textbf{Staged}
\item \textbf{Committed}
\end{itemize}

\columnbreak

git add file1 file2 dir1 dir2

git commit -m "Fancy feature done."


\end{multicols}
