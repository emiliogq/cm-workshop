\frametitle{Introduction: Git}

\begin{multicols}{2}
Git has three main states that your files can reside in:

\begin{itemize}[<+->]
\item \textbf{Modified}
\item \textbf{Staged}
\item \textbf{Committed}
\end{itemize}

\begin{lstlisting}[language=Bash]
git add file-1

git add file-2

git commit
\end{lstlisting}

\end{multicols}
